# sql-data-warehouse-project
Building a modern data warehouse with SQL Server, including ETL processes, data modelling and analytics.

Project Requirements:
Objective: Develop a modern data warehouse using SQL server to consolidate sales data,enabling analytical reporting and informed decision making.

Specification :
Data Sources : 2 sources - ERP and CRM provided as CSV files
Data Quality : Cleanse and resolve prior to analysis
Integration : Combine sources into a single use friendly data model designed for analytical queries
Documentation : Provide clear documentation to support stakeholders and analytical teams

#BI analytics and Reporting 
Objective : Develop SQL based analytics to deliver detailed insights into
1.Customer Behaviour
2.Product Performance
3.Sales Trends
